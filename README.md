## 1. Objective
Convert an existing desktop-only webpage into a mobile-friendly layout using CSS media queries.

## 2. Tools Used
- HTML5
- CSS3
- VS Code
- Chrome DevTools

## 3. Changes Made
1. Added the viewport meta tag for proper mobile scaling.
2. Implemented `@media (max-width: 768px)` for responsive design.
3. Stacked the content and sidebar vertically on smaller screens.
4. Converted the navigation menu from horizontal to vertical on mobile devices.
5. Removed fixed widths to prevent horizontal scrolling.
6. Made images responsive using:
   - `width: 100%`
   - `height: auto`
7. Tested the layout using Chrome DevTools device toolbar.

## 4. Result
- The website adapts smoothly to different screen sizes.
- No horizontal scrolling occurs on mobile devices.
- Layout remains clean and readable on smaller screens.

## 5. Outcome
Through this task, the following concepts were applied:
- Media Queries
- Responsive Web Design
- Viewport configuration
- Flexible CSS units
